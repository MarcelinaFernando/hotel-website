#  Hotel Website – Mobile Version

A hotel website developed with a focus on mobile devices, using HTML, CSS, and JavaScript.  
This project follows the **mobile-first** approach, built based on a Figma design and with dynamic rendering of repeated content using JavaScript.

---

## Introduction

This project is part of **Module 2** of the ElevateHer program.  
The goal was to build the **mobile version** of a hotel website with a focus on JavaScript, strictly following the layout provided in Figma and applying real-world development practices.

All repeated content was dynamically rendered with JavaScript, while the interface was built from scratch using semantic HTML and custom CSS.
